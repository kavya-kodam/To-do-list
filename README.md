# 📝 To-Do List App

A simple and responsive **To-Do List application** built with **React**. This application allows users to add and delete tasks ,mark has completedwhile automatically saving them using **Local Storage**, ensuring tasks remain available even after refreshing the page.

## 🚀 Features

* ✅ Add new tasks
* 🗑️ Delete existing tasks
* 💾 Store tasks using Local Storage
* 🔄 Persist tasks across page refreshes
* ⚛️ Built with reusable React components
* ✔️ Mark tasks as completed

## 🛠️ Tech Stack

* **React**
* **JavaScript (ES6+)**
* **CSS**
